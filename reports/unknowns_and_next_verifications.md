## 证据缺口总结

| 缺口 | 原因 | 后果 |
|---|---|---|
| Reddit 真实信号 | 平台 anti-bot / login wall | 轶事样本不够"独立"达到 ≥5 / ≥3 平台阈值 |
| Airbnb 单 listing 详情 | 403 阻挡 | 住房样本部分降级为 third-party guide + 价格区间 |
| Flatio 库存深度 | 站点 dynamic loading | 仅能取月租范围而非具体可订房源 |
| 清迈各区 IQAir 站点数据 | API 未开 | 全市均值已知，区差异未量化 |
| 家庭实操"3 月撤离海岛"成本 | 未纳入预算研究 | 烧林季备选方案的财务可行性需家庭自评 |
