# 里斯本 vs 清迈：家庭 3 个月慢旅基地对比研究

## 项目目标
为一个 2 大人 + 2 幼儿（约 4.5 岁 + 2.5 岁）的家庭，评估 **里斯本** 与 **清迈** 作为约 3 个月慢旅基地的可行性。

## 核心问题
两地是否能在 **签证/医疗/儿童安全/日常运营** 不引入过多摩擦的前提下，支撑一个稳定 3 个月家庭生活节奏？

## 文件夹结构
- `config/` — 调研范围、源登记、API 限速、查询模板等配置
- `data/sources/sources.csv` — 所有事实必须挂到此处的 source_id
- `data/cache/` — 所有成功的网络请求本地缓存
- `data/logs/` — 网络请求日志、限速事件、不可访问源
- `evidence/<city>/` — 各维度证据 markdown
- `reports/` — 对比矩阵、红旗、未知项、最终报告

## 关键方法论
1. **按生活功能而非景点数量比较**
2. 事实必须挂源 ID，不挂源不入正式结论
3. 轶事评论需 ≥5 条独立信号或 ≥3 平台才能成结论
4. 输出红黄绿灰 + 置信度，不输出单一总分
5. 所有产出物中文撰写；commit message 与代码注释英文
