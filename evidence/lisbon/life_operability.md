# 里斯本 — 生活可运营性（杂货 / 药店 / 洗衣 / 餐饮 / 咖啡 / 文化与自然）

## 摘要

- 主流超市连锁覆盖完整：**Pingo Doce / Continente / Lidl** 在所有候选区都有分布 [S-LIS-073][S-LIS-074]。
- 药店（farmácia）密度高，市区每 200-400m 一家（多数博客提及）；大型 hipermercado 内含 farmácia [S-LIS-073]。
- 自助洗衣：少数大型超市连锁（如 E.Leclerc）内有；中心区有 lavandaria self-service 店但样本未具体核 [S-LIS-073]。
- 餐饮：里斯本对孩子普遍非常欢迎；Cascais 沿海可在 任意时段吃 pastel/三明治 [S-LIS-035]。
- 文化 / 自然 / 旅游泡沫分隔：候选区中 **Campo de Ourique / Alvalade / Areeiro** 偏本地生活；**Belém / Cais do Sodré / Baixa** 偏旅游 [S-LIS-026][S-LIS-027][S-LIS-029][S-LIS-033][S-LIS-034]。

---

## 1. 杂货 / 超市

### 事实
- 主要连锁：**Pingo Doce、Continente、Lidl**；Continente 是国内最大零售商，多数大店内含药店 [S-LIS-073]。
- 邻里小店：mercearias（食品杂货）+ frutarias（果蔬）+ panificadoras（面包店），分布密集 [S-LIS-073]。
- 周边市场（mercados）兼具新鲜食材 + 食肆：Campo de Ourique 市场（1934 起）+ Mercado da Ribeira（Time Out Market）等 [S-LIS-026][S-LIS-074]。

### 各区供应
| 区 | 主要超市点 |
|---|---|
| Campo de Ourique | Pingo Doce（区内）+ Mercado de Campo de Ourique [S-LIS-026] |
| Estrela / Lapa | Pingo Doce Rua do Quelhas（"20 分钟步行/15 分钟公交"，靠西）[S-LIS-073] |
| Parque das Nações | **Continente Vasco da Gama**（大型 + 含药店）[S-LIS-073] |
| Belém / Restelo | Pingo Doce Belém + Continente nearby |
| Alvalade | Pingo Doce + 当地市场 + 多个 Mercearia |
| Areeiro | Pingo Doce + 周边 |
| Cascais | Continente + Pingo Doce |

### 红黄绿灰
- 杂货可达性：**Green**（confidence high）

## 2. 药店 / 健康用品

### 事实
- 葡萄牙药店（farmácia）覆盖密集 + 24h 轮值（每区每天有 farmácia de serviço）[S-LIS-073]。
- 大型超市内通常配药店；CUF / Lusíadas / Hospital da Luz 均有内部药房 [S-LIS-017][S-LIS-018][S-LIS-019]。
- 处方药 = 需医生处方；OTC 含基本儿童退烧、止咳、抗过敏可直接购买 [S-LIS-020]。

### 红黄绿灰
- 药店可获得性：**Green**（confidence high）

---

## 3. 洗衣

### 事实
- 多数长租 furnished 公寓自带洗衣机（Blueground 样本明示 2/3 含 washing machine）[S-LIS-037][S-LIS-038]。
- 烘干机较少（葡萄牙家庭多用阳台/晾衣架）→ 雨季需室内晾或自带除湿机配合 [S-LIS-046][S-LIS-081]。
- E.Leclerc 等大卖场内有自助洗衣（lavandaria self-service），中心区也有独立店但样本未具体核 [S-LIS-073]。
- 选房 checklist 已列：**洗衣机** + **除湿机**（房东配 / 自购）是雨季关键。

### 红黄绿灰
- 公寓内洗衣：**Green**（取决于选房 checklist）
- 烘干 / 雨季晾干：**Yellow**

---

## 4. 餐饮 / 咖啡

### 事实
- Pastéis de Nata 与 cafés 几乎每个街角都有；4.5/2.5 岁孩子可吃半份 + 鲜榨果汁 + 烤三明治 [S-LIS-035]。
- 葡萄牙文化对儿童在餐厅极友好 [S-LIS-035][S-LIS-064]。
- Parque das Nações 餐厅儿童套餐 ~€10–12 [S-LIS-029]。
- Cascais 海滨步道任意时段 light meal 可得 [S-LIS-035]。
- 葡萄牙午餐供应窗口约 12:00–15:00，晚餐 19:30 起；与孩子常用 17:30–18:30 节奏冲突 → cafés 是最佳折中 [S-LIS-035]。

### 红黄绿灰
- 家庭就餐友好：**Green**（confidence high）
- 时间对接：**Yellow**（与中国/北美早晚餐时间有 1–2 小时偏差）

## 5. 文化 / 自然 与日常融合

### 事实
- 中心候选区（Campo de Ourique / Alvalade / Areeiro / Estrela）= **本地家庭多于游客**，"village-like" 描述出现在多个来源 [S-LIS-026][S-LIS-027][S-LIS-033][S-LIS-034][S-LIS-064]。
- Belém + Baixa + Cais do Sodré + Alfama = **旅游泡沫**集中；Belém Tower / Jerónimos / Pastéis 高峰季拥挤 [S-LIS-032]。
- Parque das Nações = 现代化非传统街区，旅游与本地混合，相对均衡 [S-LIS-028][S-LIS-029]。
- 自然融合：Monsanto 森林 + Tagus 河滨步道 + Belém 河岸 + Cascais 海岸都在 30 分钟可达 [S-LIS-049][S-LIS-072]。

### 红黄绿灰
| 区 | 旅游泡沫 vs 本地 | 评 |
|---|---|---|
| Campo de Ourique | 本地为主 | **Green** |
| Alvalade / Areeiro | 本地为主 | **Green** |
| Estrela / Lapa | 本地为主 + 少量游客 | Green |
| Parque das Nações | 混合 | Yellow → Green |
| Belém / Restelo | 旅游热区 | Yellow |
| Cascais | 旅游 + 本地混合（淡季较好）| Yellow → Green（淡季） |

## 6. 周末 / 短途

- Sintra 火车 40 分钟（未抓但已知）。
- Algarve / Porto 需高铁/飞机半天以上。

### 红黄绿灰
- 周末微旅行：**Green**

## 7. 整体生活节奏

### 解释
- 一周典型节奏（推荐）：
  - 上午：邻里游乐场 + market 采购
  - 中午：家中或邻里 café 午餐
  - 下午（睡醒后）：室内活动（Sandcastle / Pavilhão）或室外公园
  - 周中 1-2 次跨区活动（Oceanário、Monsanto、Gulbenkian）
  - 周末 1 次跨城（Cascais / Sintra）

### 红黄绿灰
- **总评：Green**（confidence high）— 生活基础设施完整，节奏可重复。
