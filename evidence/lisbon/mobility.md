# 里斯本 — 移动性（步行 / 公交 / 打车 / 婴儿车 / 儿童安全座椅）

## 摘要

- 公交无障碍程度较高：**Metro > 75% 站点无台阶** + Carris 公交 > 80% 装有低地板斜坡 [S-LIS-067][S-LIS-068]。
- 婴儿车在中心老区遇到的最大摩擦是**山坡 + 卵石路 + 老楼无电梯**，多个家庭博客一致建议改用 baby carrier 或在某些日子用车 [S-LIS-044][S-LIS-045][S-LIS-069]。
- 儿童安全座椅法律：**12 岁以下或身高 < 135cm 必须使用座椅**；3 岁以下不得坐前排 [S-LIS-070][S-LIS-071]。
- 出租车 / Uber / Bolt **法律豁免**儿童座椅要求；但安全推荐自带或预约带座椅服务（Taxi Bambino 等）[S-LIS-070][S-LIS-071]。
- Lisbon → Cascais 火车：40 分钟 + €2.55/成人、€1.30/儿童 + 4 岁以下免费 → 周末海滩可达 [S-LIS-067][S-LIS-072]。

## 1. 步行

### 事实
- 多个家庭博客一致提到：里斯本中心多山坡 + cobblestone（calçada）人行道 → 不平整、对婴儿车不友好 [S-LIS-044][S-LIS-045]。
- Parque das Nações 是少数全区平坦 + 现代化人行道的候选区 [S-LIS-028][S-LIS-029]。
- Belém / Cascais 河边/海边步道平坦 [S-LIS-032][S-LIS-035]。
- 推荐工具：Google Maps 选"轮椅无障碍"路线，可避开主要楼梯/斜坡 [S-LIS-069]。

### 红黄绿灰
| 区 | 步行/婴儿车友好度 |
|---|---|
| Parque das Nações | **Green** |
| Belém / Restelo | Green（河边）/ Yellow（Restelo 内部坡） |
| Cascais / Estoril | **Green** |
| Campo de Ourique | Yellow（中心平台，但出区有坡） |
| Alvalade / Areeiro | Green / Yellow |
| Estrela / Lapa | **Yellow → Red**（依房而定，老城坡多） |

## 2. 公共交通（Metro / Carris bus / Tram）

### 事实
- **Metro**：≥ 75% 站点无台阶（电梯/斜坡/宽闸门）；设有为婴儿车保留的特殊闸口，必要时由站务员协助 [S-LIS-067]。
- **Carris bus**：几乎全部 city bus 为 "kneeling buses" 配可伸缩斜坡 + 残障/婴儿车专区；> 80% 配斜坡 [S-LIS-068]。
- Alfama 区的微型公交（minibus）小且无斜坡，**不适合婴儿车** [S-LIS-069]。
- **4 岁以下儿童**免费乘坐所有公交 [S-LIS-069]。
- 家庭友好公交线路推荐：**714、727**（途经景观街区，相对不挤）[S-LIS-069]。
- 复古 tram（12 / 28 / 25 / 15）非常拥挤 + 婴儿车难以登车 + 是**扒手高发线**；旅游博客明确建议家庭"避开高峰" [S-LIS-077][S-LIS-078]。

### 红黄绿灰
- Metro 婴儿车友好：**Green**
- 公交可达性：**Green**
- Tram 历史线路（28 等）：**Yellow / Red**（拥挤 + 扒手 + 婴儿车难，建议作为景点而非日常）

## 3. 出租车 / Uber / Bolt / FreeNow

### 事实
- Uber、Bolt、FreeNow 在里斯本运营良好，覆盖核心区 + Cascais [S-LIS-070][S-LIS-071]。
- 法律：**出租车 + ride-share 豁免儿童座椅要求**（3 岁以下可不约束，3 岁以上需坐后排系安全带）[S-LIS-070][S-LIS-071]。
- 操作上 Uber 等可在 booking 时备注"需要婴儿座椅"，但**不保证**实际车辆有 [S-LIS-070]。
- **Taxi Bambino** 等专门家庭出租车服务可预订含座椅 [S-LIS-070][S-LIS-071]。

### 解释
- 与加拿大法律对比，本地 ride-share 豁免座椅是显著松动点；从家庭风险容忍（低-中）角度看，建议：
  - 短途同城 Uber：可豁免（孩子坐后排 + 大人抱 / 系安全带，2.5 岁建议大人抱紧）
  - 机场接送 / 长途 / 跨城：**预约 Taxi Bambino 等含座椅服务**
  - 或自带 2 个轻便升级座椅（4.5 岁可用 booster，2.5 岁需 group 1）[S-LIS-070][S-LIS-071]。

### 未知 / 下一步
- 自带座椅在 Uber 安装的实操（部分车型安全带配置不兼容）→ 到达后试验。
- Bolt vs Uber 价格差（一般 Bolt 略低）。

### 红黄绿灰
- 打车可获得性：**Green**
- 儿童座椅合规：**Yellow**（法律松、需主动准备）

## 4. 私人租车 / 长租

### 事实
- 私人车辆 + 租赁汽车**强制座椅** [S-LIS-070][S-LIS-071]。
- 3 个月慢旅本家庭对长租车需求低（中心区步行 + 公交 + 短途 Uber 已可覆盖）。

### 红黄绿灰
- 不依赖私车：**Green**（确认 candidate 区均无私车需求）

## 5. 周末跨城

### 事实
- **Lisbon → Cascais**：Cais do Sodré 站，40 分钟，€2.55/€1.30/张（单程），Navegante 卡 €0.50/张 [S-LIS-072]。
- 火车沿海岸线，景观佳；Cascais 是无坡走 town（与里斯本对比）[S-LIS-035][S-LIS-072]。
- 4 岁以下免费 [S-LIS-069]。
- **Lisbon → Sintra**：火车从 Rossio 出发，约 40 分钟（未抓但已知）。

### 红黄绿灰
- 周末跨城可达性：**Green**

## 6. 综合建议

| 用途 | 推荐 |
|---|---|
| 邻里日常 | 步行 + Carris bus（无障碍线 714/727） |
| 中心区景点 | Metro（婴儿车闸口）+ 普通公交 |
| 紧急医疗 | 自家附近私立 24h 儿科步行 / Uber < 10 分钟 |
| 雨天 | Uber / Bolt（避免雨中拖婴儿车上坡） |
| 跨城 / 海滩 | CP 火车（Cascais 线 / Sintra 线）|
| 机场接送 / 长途 | Taxi Bambino 含座椅 |

## 红黄绿灰 — 总评
- 公交无障碍度：**Green**（confidence high）
- 中心区步行婴儿车摩擦：**Yellow**（confidence high）— 老城有坡 + 卵石，但有 baby carrier + Uber + Metro 互补
- 儿童座椅合规风险：**Yellow**（confidence high）— 法律松，需自带
- 总评：**Green**（confidence medium-high）
