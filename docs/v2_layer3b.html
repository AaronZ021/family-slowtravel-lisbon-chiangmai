范围：indicators 3.8–3.14 + 3.18–3.20 目标家庭：2 大人 + 4.5 岁 + 2.5 岁 输出原则：每个 indicator 必须有官方 / 平台 indices + ≥3 平台或 ≥5 来源社区佐证 + 判断 + 置信度。不输出城市总评。 新增源：LIS S-LIS-104 → S-LIS-120 (17 条)；CMI S-CMI-077 → S-CMI-090 (14 条)。

Indicator 3.8 — 步行可达度 (Walkability)¶

定义/重要性：90 天慢旅家庭日常生活半径 1 km 内能否完成核心 chores（杂货 / 药店 / 游乐场 / 24h 儿科），步行强度对 2.5 岁幼儿决定每天能否出门。

期望数据源：Walk Score / 全球 walkability 排名 / 城市规划报告 / Numbeo Traffic + Pollution / 社区一致评分。

充分性阈值：每候选区 1km 内 4 类设施可达数 + 全球 walkability 排名 + ≥3 来源社区一致。

城市 主要统计数据点 平台 indices 社区佐证 判断 置信度 关键源
里斯本 9th 全球 most walkable（Compare the Market 90 城排名）；84% 居民步行可达学校 / 医疗（top-10 中最高）[S-LIS-109] Numbeo Traffic Commute 34.64（低）；Pollution 37.55（低）[S-LIS-104] 多源指出 Parque das Nações / Belém / Cascais 完全平坦；Alfama / Bairro Alto 七山 + calçada 卵石 → 婴儿车坡度负担 [S-LIS-118][S-LIS-044][S-LIS-045][S-LIS-049]；CUF Descobertas 儿科在 Parque das Nações 步行可达 [Lisbon search] Green（候选区 Parque das Nações / Belém / Cascais）/ Yellow（Alfama / Bairro Alto / Estrela 内部坡） high S-LIS-109, S-LIS-104, S-LIS-118, S-LIS-044, S-LIS-049
